## 01
```
// step 1 - Инициализация счетчика `int a=0;` Инициализирует новую переменную и обозначает начальное значение переменной управления циклом
// step 2 - Условие `x<10;` Логическое выражение для проверки значения переменный управления циклом
// step 3 - Изменение счетчика `System.out.println("a value is:" +a);` Если результат условия - `true`, выполняется цикл
// step 4 - Итерация `a=a+1` Определяет порядок изменения переменной управления циклом на каждом его шаге
for(int a=0; a<10; a=a+1){
	System.out.println("a value is:" +a);
}
```

// можно объявить переменную в самом цикле,но область видимости переменной ограничется циклом
for (int a = 1; a<5; a++) {

## 01
```
// можно объявить несколько переменных и несколько операций в части итерации для каждой переменной
for (int a = 1, b = 4; a<b; a++, b--) {  
	System.out.println("a =" + a);  
	System.out.println("b =" + b);  
}
```

## 01
```
// инициализирующее и итерационное выражения вынесены за пределы цикла `for`
// выполнение цикла продолжается,пока в переменной done не установиться логическое значение `true`
// условием,управляющем циклом,может быть любое логическое выражение,а не только условное выражение 
boolean done = false;  
int i = 0;  

for (; !done; ) {  
    System.out.println("i = " + i);  
 if (i == 10) done = true;  
 i++;  
}
```

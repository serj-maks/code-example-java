## 01
```
int n;  
n = 2;  
switch (n) {  
    case (1):  
        System.out.println("1");  
    break;  
// case может содержать несколько вариантов, при попадании в которые
// будет возвращаться один результат 
    case (2):  
    case (3):  
        System.out.println("2");  
    break;  
 
    default:  
        System.out.println("default block");  
    break;
 }
```

## TreeSet methods
### first()
```java
// выводит первый по списку элемент коллекции
treeSet.first();
```

### last()
```java
// выводит первый по списку элемент коллекции
treeSet.last();
```

### headSet();
```java
// выводит элементы из головы, НЕ ВКЛЮЧАЯ вводимое значение
treeSet.headSet(2);
```

### tailSet();
```java
// выводит элементы из хвоста, ВКЛЮЧАЯ вводимое значение
treeSet.tailSet(4);
```

### subSet();
```java
// выводит элементы между заданными
treeSet.subSet(1,4);
```
